# Wind & Solar Daily Power Production

🚀 **Project Overview**  
This project analyzes daily power production from **wind** and **solar** sources. Using Python and real-world datasets from Kaggle, the project explores trends, patterns, and insights to support smarter renewable energy planning.

---

## 📂 Dataset
The dataset contains daily energy production data for wind and solar power. It includes:
- Date
- Power production (MW)
- Source (Wind or Solar)
- Additional features for analysis

**Source:** [Kaggle Notebook](https://www.kaggle.com/code/usmanshafeeqdit/wind-solar-daily-power-production)

---

## 🛠️ Tools & Libraries
- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

---

## 📊 Analysis & Visualizations
1. **Data Cleaning & Exploration**
   - Checked for missing values and duplicates
   - Explored descriptive statistics
2. **Trend Analysis**
   - Daily and monthly production trends
   - Seasonal patterns in wind and solar energy
3. **Comparative Analysis**
   - Wind vs Solar production over time
   - Correlation and insights for energy planning
4. **Visualizations**
   - Line plots for trends
   - Bar charts for comparisons
   - Interactive plots using Plotly

---

## 💡 Key Insights
- Solar power shows higher production in summer months, while wind production peaks in winter.
- Combined wind and solar output can provide a more stable energy supply.
- Insights from trends can assist in planning and optimizing renewable energy grids.
